## Clean and Seamless without Title

```json
{
  "primary-theme@@embed-background@@dark": "#00000000",
  "primary-theme@@embed-background@@light": "#00000000",
  "primary-theme@@embed-hide-title": true,
  "primary-theme@@embed-border-top-width": "0px",
  "primary-theme@@embed-border-end-width": "0px",
  "primary-theme@@embed-border-bottom-width": "0px",
  "primary-theme@@embed-border-start-width": "0px"
}
```

## Clean and Seamless with Title

## Round Tabby with Thick Left Border

```json
{
  "primary-theme@@embed-title-border-width": "0px",
  "primary-theme@@embed-border-radius": "0px 12px 12px 0px",
  "primary-theme@@embed-border-top-width": "0px",
  "primary-theme@@embed-border-end-width": "0px",
  "primary-theme@@embed-border-bottom-width": "0px",
  "primary-theme@@embed-border-start-width": "3px"
}
```

## Tiny Uppercased Note Embed Title

```json
{
  "primary-theme@@embed-title-font-size": "10px",
  "primary-theme@@embed-title-font-weight": 450,
  "primary-theme@@embed-title-text-transform": "uppercase",
  "primary-theme@@embed-title-letter-spacing": "1px",
  "primary-theme@@embed-title-border-style": "solid"
}
```

The preset also works with `dashed`, or `dotted` Embed Title Border Style.
